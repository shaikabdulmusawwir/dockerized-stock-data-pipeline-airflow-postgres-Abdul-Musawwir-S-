# dockerized-stock-data-pipeline-airflow-postgres-Abdul-Musawwir-S-
“Dockerized data pipeline using Apache Airflow and PostgreSQL to automatically fetch stock market data from a public API (e.g., Alpha Vantage), parse the JSON response, and store it in a Postgres table on a schedule. Includes error handling, environment‑based secrets, and one‑command startup via Docker Compose.”
